v4.0.0
一、实现该软件与FPGA的UDP通信，接收信号发生器发送的正弦波信号。最大传输速率为10MBps。不存在丢包现象。
二、能将正弦波信号保存到本地，存储格式为ASCII和HEX两种。支持1分钟存储一个文件，并命名为当前时间。
三、能将本地的HEX格式的正弦波信号文件还原成正弦波并显示在窗口。

v4.2.0
1.能将FPGA发送的一个1024数据包以波形显示在窗口
2.32帧数据对应采集卡采集的4096个数据点，能实现每次显示32帧数据在窗口。使用信号和槽方式发送和接收数据。
3. 同时显示2048点会很卡，建议显示512点

v4.2.3
1. 加入了解调功能，还未进行测试

v4.3.0
1.添加功能模式二，接收FPGA发送过来的峰值点（PeakVal），解析成三通道数据，然后保存到本地或者解调。
2.添加串口发送本地配置及文件的功能。

v4.4.0
1. 添加了甄别起始数据帧的功能。采集卡采集的4096个点用32帧以太网帧发，在首帧中间128~144处增加了00000000（化为HEX为3030303030303030）作为鉴别位。
2. 识别出首帧后，将从首帧开始的32帧数据显示在wave_widget上（显示长度为原始数据的1/4），称为一个周期。
3. 可以设置显示的刷新频率，比如每10个周期显示一次。
